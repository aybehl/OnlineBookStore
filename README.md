# DATABASE COLLECTIONS
Books
- Id
- name
- author_name
- price
- quantity
- cover_picture
- summary

# ROUTES

Render Homepage
GET /

Render Store
GET /store

Get all the books from DB
GET /books

Get a specific Book
GET /books/:id

Admin Page containing form to add a new book
GET /admin/books/add

Admin can add a new book
POST /admin/books/add
